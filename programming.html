C# Development Work

School Statistics Program

I was the sole developer on a university group project where we created a software package to be used by a certain north east secondary school. Multiple teams competed against eachother and wrote programs and the one I wrote won best program overall. The program had to be able to import school data output that was saved as xlsx and then be able to manipulate it so the school could retrieve statistical data and charts. The program would convert a 100+ column excel document into individual arrays which could then be searched for and manipulated through the program.

Due to the nature of who the program was made for this code can't be shared
